The pipeline

Three steps, one Lambda function

01

Upload

Your image is read in the browser and sent as base64 to an API Gateway endpoint — no accounts, no plugins.

02

Resize

A Lambda function resizes and compresses the image to the resolution you picked, on the fly.

03

Store & serve

Both versions land in S3, and you get back links, dimensions, and exactly how much space you saved.
